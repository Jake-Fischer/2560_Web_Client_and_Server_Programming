Netflix Login and Pynput Fun

This is a little project I was working on for an hour or two, mostly to try to learn about pynput, which lets you control the mouse and keyboard of a computer with python code

In this example, I use it to log into netflix from a browser (login info removed from examples of course). There might be practical applications for this to automate tasks with websites that have bad/no APIs, but it would have serious limitations.

I think its a cool package though, and a cool example of what python can do.


        import pynput

        from pynput.mouse import Button, Controller
        from pynput.keyboard import Key
        from pynput.keyboard import Controller as KeyController
        import time

        mouse = Controller()
        keyboard = KeyController()

        original_position = (mouse.position[0], mouse.position[1])
        username_position = (2820, 352)
        password_position = (2879, 417)
        clear_position = (3067, 386)
        sign_in_position = (2874, 496)

        mouse.position = (username_position[0], username_position[1])
        time.sleep(1)
        mouse.click(Button.left, 3)
        time.sleep(1)
        keyboard.press(Key.backspace)
        keyboard.release(Key.backspace)
        time.sleep(1)
        keyboard.type('')
        time.sleep(1)
